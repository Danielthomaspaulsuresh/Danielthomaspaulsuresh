## IT Thesis

### AI-Driven Bioacoustic Analysis for Phylogeographic Mapping of the Pink Cockatoo
Developed an end-to-end AI pipeline integrating data acquisition, spectrogram
generation, feature extraction, and supervised/unsupervised modelling to
analyse regional vocal dialects of the Pink Cockatoo across Northern
Australia. Trained lightweight CNN and CRNN architectures on 698 spectrogram
samples, improving regional dialect classification and revealing spatial and
temporal acoustic patterns. Conducted spatial mapping and ecological
interpretation by linking model outputs with IBRA bioregions, supporting
conservation insights through phylogeographic trend analysis.

`Python` `CNN` `CRNN` `Machine Learning` `Bioacoustics` `GIS`
